2.13. Metadata Formats¶

These formats are used for the Metadata Interface interface only.

  • 2.13.1. V4L2_META_FMT_D4XX (‘D4XX’)
  • 2.13.2. V4L2_META_FMT_IPU3_PARAMS (‘ip3p’), V4L2_META_FMT_IPU3_3A (‘ip3s’)
  • 2.13.3. V4L2_META_FMT_RK_ISP1_PARAMS (‘rk1p’), V4L2_META_FMT_RK_ISP1_STAT_3A (‘rk1s’)
  • 2.13.4. V4L2_META_FMT_UVC (‘UVCH’)
  • 2.13.5. V4L2_META_FMT_VSP1_HGO (‘VSPH’)
  • 2.13.6. V4L2_META_FMT_VSP1_HGT (‘VSPT’)
  • 2.13.7. V4L2_META_FMT_VIVID (‘VIVD’)
